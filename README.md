# Aplikasi To-Do List

# Deskripsi 
Aplikasi sederhana berbasis PHP untuk mencatat tugas harian

# Fitur
1. Tambah tugas
2. Tandai tugas selesai
3. Edit tugas
4. Delete tugas

# Struktur Folder
1. 'index.php' 
2. 'style.css'
3. 'data.php' 
4. 'function' 

# Cara Menjalankan 
1. Salin folder ke 'htdocs/'
2. Jalankan XAMPP dan buka di'http://localhost/todolist/index.php'
